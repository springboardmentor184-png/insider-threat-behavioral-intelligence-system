# Insider Threat Behavioral Intelligence System

## Project Objectives

### Primary Objective

Develop an AI-powered cybersecurity platform capable of monitoring employee behavior, detecting insider threats, and assisting security analysts through intelligent risk analysis.

---

## Specific Objectives

- Monitor employee login activities
- Track file access behavior
- Record application usage
- Detect abnormal user behavior
- Calculate dynamic insider risk scores
- Generate real-time security alerts
- Provide incident investigation support
- Visualize employee activity using dashboards

---

## Technologies

- FastAPI
- Python
- SQLAlchemy
- SQLite (Development)
- React
- Machine Learning
- JWT Authentication

---

## Expected Outcome

A secure behavioral intelligence platform capable of identifying suspicious insider activities using User and Entity Behavior Analytics (UEBA).
